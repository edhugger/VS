Игру можно скачать по ссылке: https://github.com/edhugger/VS
Для запуска приложения необходимо открыть файл «Теория симуляции.exe», расположенный в папке «Теория симуляции», после чего выводится меню с выбором дальнейших действий в игре.
Для начала новой игры необходимо нажать кнопку «Начать». 
Сцены переключаются с помощью щелчка мышки. 
На экране с выбором ответов, нужно навести мышку на ответ и щелкнуть.
Для сохранения игрового процесса необходимо нажать кнопку «Сохранить» и выбрать место сохранения.
Рисунок 30 – Сохранение игрового процесса
Если уже есть сохраненный процесс, тогда в главном меню необходимо нажать кнопку «Загрузить» и ячейку с нужным сохранением.
Для того, чтобы вернуться на прошлую сцену придется нажать кнопку «Назад» на нижней панели.
Бои в игре осуществляются с помощью щелчка мышки по врагу.
